Two instrument modes — toggle between ✦ Choir and ♩ Strings. Each mode has 4 voice presets:

Choir: Angelic · Cathedral · Ethereal · Soprano
Strings: Celestial · Lush · Tremolo · Pizzicato

The sound engine is built for lushness — each note spawns multiple detuned oscillators with per-voice vibrato LFOs that swell in naturally after key-press, plus a noise layer for breath texture in choir mode. A cathedral-scale convolution reverb (4.5s impulse) wraps everything in space.
Ensemble harmony (UNI / OCT / 5TH) layers a second voice at the octave or perfect fifth automatically. The long attack/release defaults mirror how real choirs and string sections breathe.
The voice canvas shows an animated rose curve (4-petal for choir, 3-petal for strings) with harmonic rings and floating motes that activate more intensely when notes are held. Pressing keys also spawns golden/celestial shimmer particles above the keyboard.
